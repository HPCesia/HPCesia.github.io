功能

这节推荐的都是比较通用的功能性扩展，针对某个语言的扩展也不需要我来推荐，自有大把针对性的文章进行介绍。

Path Intellisense

Path Intellisense 扩展可以在输入文件路径的时候给出自动补全提示，方便输入。

Project Manager

Project Manager 扩展可以在侧边栏新增一个「项目管理器」选项，通过在项目管理器中保存项目，可以实现不同工作区之间的快速切换。

Error Lens

Error Lens 扩展可以直接把错误、警告与提示等信息直接显示在代码行后面，而不是只能把光标移动到下划线处才能看到。

Todo Tree

Todo Tree 扩展可以搜索整个工作区内的「TODO」、「FIXME」、「BUG」等字段，并在侧边栏的选项中查看。

Trailing Spaces

Trailing Spaces 扩展可以标红每行代码尾部多余的空格。

Excalidraw

Excalidraw 扩展可以在 VSCode 内绘制手绘风格的矢量图片，支持导出 .png 和 .svg 格式。

CodeGeeX

CodeGeeX 扩展可以调用本地或网络的大模型，来加速代码编写。CodeGeeX 的大模型在 GitHub 上开源，因此可以通过本地运行大模型，放心地在涉及商业秘密的项目中使用。

Continue

Continue 同样是使用大模型进行代码补全、提供编辑器内对话的扩展，不同的是 Continue 支持更多的大模型选择，可配置的项目也更多。界面都是英文，但是看懂难度不大。

Code Spell Checker

Code Spell Checker 扩展可以检测代码中的英文拼写错误并给出修改建议。

翻译(英汉词典)

翻译(英汉词典) 扩展可以在选中单词时在右下角给出单词的翻译，并且可以翻译下划线、驼峰等格式的单词，方便理解代码。

效果演示

驼峰翻译助手

驼峰翻译助手扩展可以调用翻译 API，将中文直接翻译为英文的不同格式的变量名，再也不用发愁怎么命名变量。

美化

Material Icon Theme

Material Icon Theme 扩展是一个文件图标扩展，其为各种文件与文件夹都制作了特殊的图标，方便浏览。

background-cover

background-cover 扩展可以直接在其自带的命令中修改更换 VSCode 背景图片。

SpaceBox UI Enhancer

SpaceBox UI Enhancer 扩展为 VSCode 的菜单增加了大量对效率影响较小的过渡动画效果。

评论区反应 MacOS 下该插件可能不可用，请注意。
