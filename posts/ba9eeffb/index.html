准备工作

由于我们最终是要使用 VSCode 而非 Visual Studio 进行 C++ 开发，因此本节 Clang 的安装将使用 MSYS2 + mingw-w64 进行。

安装 MSYS2

在 Github Release 上下载最新版的 MSYS2 安装包。如果访问 Github 有困难，也可以使用清华镜像源进行下载。

下载后打开安装程序，选择一个合适的路径，然后一路点 next 即可安装完成。
